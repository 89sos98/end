End.js
===============
## What
End.js 是基于Node.js和MongoDB的Baas系统，旨在成为一个可方便部署到Node.js的Package。
## Why
想StackMob、Parse、Firebase目前都提供非常好的Baas服务，但是他们都有一些缺点
+ 不可以部署到自己的机器上
+ 当然也不可以设置自己的数据库

这些缺点导致项目会面临

+ 数据安全
+ 性能无法自行优化，只能依赖第三方

## Features
+ REST data
+ RealTime
+ Status Control
+ Easy Push
+ More Apis
