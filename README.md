NodEnd

# NodEnd

这是我的毕设项目，毕设的题目是基于Node.js和MongoDB的BaaS系统的设计与实现
简单的讲是一个提供数据库CRUD RESTful API、数据访问控制、可插件扩展、服务器 数据库状态查询 可一键部署的Node.js的Package。
毕设期间除了提供基础功能外，还会提供Web SDK
毕设结束会继续维护，也希望大家可以一起加入，并协助我开发多个平台的SDK
